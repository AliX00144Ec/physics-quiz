# Physics Quiz - Bac 2026 🎓
تطبيق تفاعلي لمراجعة دروس الفيزياء والكيمياء للسنة ثانية باكالوريا علوم فيزيائية.

## المميزات:
- أكثر من 150 سؤال (قيد التطوير لـ 1000).
- تصحيح تلقائي فوري بالألوان.
- مقسم حسب الوحدات (موجات، نووي، كهرباء...).
- واجهة متوافقة مع الهاتف.

## التقنيات المستخدمة:
- React.js
- Vite
- Vercel (Deployment)
